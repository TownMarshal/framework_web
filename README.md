## 使用

### clone

```bash
$ git clone https://github.com/iczer/vue-antd-admin.git
```

### yarn

```bash
$ yarn install
$ yarn serve
```

### or npm

```
$ npm install
$ npm run serve
```

更多信息参考 [使用文档](https://iczer.gitee.io/vue-antd-admin-docs)
