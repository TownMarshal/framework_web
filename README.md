## 使用

### clone

```bash
$ git clone https://github.com/iczer/vue-antd-admin.git
```

### yarn

```bash
$ yarn install
$ yarn serve
```

### or npm

```
$ npm install
$ npm run serve
```

## 建议使用vscode编辑器

安装下列插件实现统一格式化

```
安装插件 Prettier 
安装插件 Vetur 
设置 vue 默认格式化插件为 Vetur
其他文件格式选择 Prettier
```
